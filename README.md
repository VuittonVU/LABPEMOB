# 2. Project Description
AnimeVerse merupakan aplikasi katalog anime yang berfungsi sebagai platform untuk menelusuri dan mengelola daftar anime, mirip dengan layanan seperti MyAnimeList. Aplikasi ini dibangun dengan memanfaatkan Flutter sebagai framework front-end, serta Firebase Authentication dan Firestore untuk fitur autentikasi dan penyimpanan data. Fitur aplikasi mendukung:
- Login menggunakan Email dan Password
- Login menggunakan akun Google
- Forgot Password dan Change Password
- Pencarian anime berdasarkan kata kunci
- Filter anime berdasarkan genre
- Fitur “Load More” untuk pagination
- Penambahan dan penghapusan anime dari daftar favorit
- Halaman profil pengguna
- Logout
- Integrasi penuh dengan Firebase Console untuk keperluan autentikasi dan penyimpanan data pengguna
